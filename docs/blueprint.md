# **App Name**: Indian Education Exam Cell, India

## Core Features:

- Organization Registration: Allow schools, colleges, universities, and government bodies to register with official email verification.
- User Authentication: Secure login for registered organizations.
- Syllabus Upload and Management: Enable authorized users to upload syllabus and notes for each class and subject.
- Question Paper Generation: Generate question papers based on class, marks, syllabus, and organization type.
- AI-Powered Question Algorithm: Implement an AI algorithm that chooses appropriate questions with different cognitive complexity. It uses syllabus details, Bloom's Taxonomy, course outcomes, and mark distribution as its tool.
- Chief Exam Cell Officer Dashboard: Dashboard for the Chief Exam Cell Officer to manage registered organizations and generated question papers.
- Question Paper Export: Export generated question papers in .docx format according to the registered organization and authority.

## Style Guidelines:

- Primary color: Indian Saffron (#FF9933) to reflect the Indian identity.
- Background color: Very light grey (#F0F0F0) to provide a clean and professional look.
- Accent color: Forest Green (#228B22) for complementary contrast, also symbolizing growth and education.
- Headline font: 'Poppins', a geometric sans-serif, for titles and important information.
- Body font: 'PT Sans', a humanist sans-serif, for the body of the application and general text.
- Use clear and professional icons related to education and examination.
- Clean, intuitive layout with easy navigation and a professional design.
